## Features

- Balanced, minimal dark palette (`#1A1C2E`, `#242933`, `#0D0F10`)
- Clean syntax highlighting for major languages
- Minimal UI distractions for immersive focus
- Smooth contrast for readability

---

## Installation

**Extensions Panel**
1. Open **VSCodium** or **VS Code**
2. Go to the **Extensions** panel (`Ctrl + Shift + X`)
3. Search for `KAROU Theme`
4. Click **Install**

**Command Line**
```bash
ext install KAROU Theme
```
